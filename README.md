# 🐉 Universo Dragon Ball Games

Proyecto desarrollado para la asignatura de Diseño Web.

## 📖 Descripción

Universo Dragon Ball Games es un sitio web estático dedicado a algunos de los videojuegos más populares de Dragon Ball. El proyecto permite conocer información sobre la franquicia y explorar dos de sus títulos más destacados:

- Dragon Ball Sparking! ZERO
- Dragon Ball Xenoverse 2

El sitio fue desarrollado utilizando HTML5 y CSS3, aplicando principios básicos de diseño web, navegación entre páginas, contenido multimedia y enlaces externos.

---

## 🎮 Páginas del Sitio

### 🏠 Inicio
- Presentación del proyecto.
- Información general sobre Dragon Ball.
- Imagen de Goku.
- Video relacionado con Dragon Ball.
- Enlace al sitio oficial.

### ⚡ Dragon Ball Sparking! ZERO
- Descripción del videojuego.
- Características principales.
- Imagen promocional.
- Video oficial.
- Enlace al sitio oficial.

### 🌟 Dragon Ball Xenoverse 2
- Información del videojuego.
- Características destacadas.
- Imagen promocional.
- Video oficial.
- Enlace al sitio oficial.
